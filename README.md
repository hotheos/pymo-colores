# Pymo — Guía de Marca y Colores Oficiales

Página web con la paleta de colores oficial de Pymo, diseñada para que diseñadores y socios externos puedan consultar y copiar fácilmente los códigos de color del logo.

## Colores de la marca

| Color | Pantone | CMYK | HEX |
|---|---|---|---|
| Azul Cielo | 7688 C | 70, 28, 4, 0 | #4597CB |
| Azul Profundo | 646 C | 65, 40, 15, 0 | #6489B1 |
| Negro Pymo | Black C | 70, 65, 65, 65 | #2C2B2A |
| Blanco | 000 C | 0, 0, 0, 0 | #FFFFFF |

## Funcionalidades

- Visualización de los 4 colores corporativos con sus códigos en Pantone, CMYK y HEX
- Copia individual de cada código al portapapeles
- Copia masiva de todos los colores en un formato
- Degradado del isotipo con código CSS copiable
- Información de contraste WCAG entre combinaciones de colores
- Descarga directa del logo en PNG

## Ver la página

La página está publicada en GitHub Pages:

**https://hotheos.github.io/pymo-colores/**
